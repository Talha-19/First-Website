My Projects

Down below are the names of some of the projects that I have worked on and completed. Click on the project name to find out more about the project and view a snippet of how it looked.

Number Guesser Project

In this project, I wrote JavaScript functions to power a small guessing game where the user has to choose a number between 1-9 and the computer also randomly chooses a number. Whoever's guess is the closest to the target number wins.

Rock Paper Scissors Project

In this project, I created the classic Rock Paper and Scissors game. To play this game, the user must choose from the following: Rock, Paper, or Scissors. Once the user has decided, it shows the computer's choice and decides the winner based on the two choices. If the user and the computer both choose the same option, the game ends in a draw.

Fotomatic Project

In this project, I was provided with a website that had text and images all over the place. I had to use CSS styles, especially flexbox, to make everything look neat and in place.
